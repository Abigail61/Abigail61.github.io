2. 集群上使用nvidia-smi指令

如果登录了服务器后，直接在命令行中输入nvidia-smi，会有如下报错：
bash: nvidia-smi: command not found
这是因为在集群中，我们只是登录到服务器上了，但没有运行作业，没有分配到GPU。我们需要提交一个作业，并在作业中运行nvidia-smi指令，从输出文件中读取相关信息。
